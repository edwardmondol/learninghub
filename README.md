# Modern To-Do App

A polished to-do app built with React, Vite, and TailwindCSS. Create, edit, and delete tasks with descriptions, mark them complete, and filter your list. Tasks persist in your browser via `localStorage`.

## Features
- Add tasks with a title and optional description
- Edit tasks inline
- Delete tasks
- Toggle complete / active
- Filter: All / Active / Completed
- Animated gradient background and glassmorphism UI
- Auto-saves to `localStorage`

## Getting Started

```bash
npm install
npm run dev
```

Then open the URL shown in the terminal (usually http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```
